# 应用图标

请在此目录添加以下图标文件：

## 需要的图标

| 文件名 | 尺寸 | 格式 | 说明 |
|--------|------|------|------|
| icon-192.png | 192x192 | PNG | 主图标 |
| icon-512.png | 512x512 | PNG | 大图标 |
| icon-maskable.png | 512x512 | PNG | Maskable图标（留出安全边距） |

## 图标设计建议

- 简洁的文档/编辑相关图标
- 背景色: #007aff 或白色
- 使用SVG或高分辨率PNG源文件

## 在线生成工具

- https://www.pwabuilder.com/imagegenerator
- https://realfavicongenerator.net/
- https://romannurik.github.io/AndroidAssetStudio/

## 临时方案

在正式图标准备好之前，可以：
1. 使用纯色占位
2. 使用在线工具快速生成
3. 从开源图标库选择（如Material Icons）
